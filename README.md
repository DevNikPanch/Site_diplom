---

## 📋 Описание

Веб-приложение предоставляет единую цифровую среду для взаимодействия абонентов с ресурсоснабжающей организацией. Система заменяет устаревшие очные способы обслуживания и позволяет клиентам дистанционно управлять лицевыми счетами, передавать показания приборов учёта, подавать заявления, а сотрудникам – оперативно обрабатывать обращения.

Проект выполнен на базе фреймворка Django с использованием классического стека HTML/CSS/JS и протестирован с помощью Selenium WebDriver.

## ⚙️ Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/yourusername/ulvdk.git
cd ulvdk
```

### 2. Создание виртуального окружения
```bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate     # Windows
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения (опционально)
Создайте файл `.env` в корне проекта и укажите чувствительные данные:
```ini
SECRET_KEY=ваш_секретный_ключ
EMAIL_HOST_USER=your_email@yandex.ru
EMAIL_HOST_PASSWORD=пароль_приложения
```

### 5. Применение миграций
```bash
python manage.py migrate
```

### 6. Создание суперпользователя
```bash
python manage.py createsuperuser
```

### 7. Загрузка начальных данных (тарифы, тестовая статья)
```bash
python manage.py loaddata initial_data
```
Если фикстура отсутствует, создайте тарифы через админ-панель (ХВС 45.50 руб/м³, водоотведение 32.20 руб/м³).

### 8. Запуск сервера разработки
```bash
python manage.py runserver
```
Приложение будет доступно по адресу `http://127.0.0.1:8000`.

---

> Для работы Selenium требуется установленный ChromeDriver. Библиотека `webdriver-manager` загрузит его автоматически.
